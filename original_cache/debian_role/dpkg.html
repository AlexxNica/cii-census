Debian package management system

This package provides the low-level infrastructure for handling the installation and removal of Debian software packages.

For Debian package development tools, install dpkg-dev.

Tags: System Administration: Package Management, Implemented in: C, implemented-in::perl, interface::commandline, Role: Program, Scope: Utility, Application Suite: suite::debian, use::configuring, Works with: Packaged Software
