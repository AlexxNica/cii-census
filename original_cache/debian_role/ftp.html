classical file transfer client

This is the user interface to the ARPANET standard File Transfer Protocol. The program allows a user to transfer files to and from a remote network site.

Tags: Implemented in: C, User Interface: Command Line, interface::text-mode, network::client, Network Protocol: FTP, Role: Program, Interface Toolkit: uitoolkit::ncurses, use::downloading, Works with: Files
