Apply a diff file to an original

Patch will take a patch file containing any of the four forms of difference listing produced by the diff program and apply those differences to an original file, producing a patched version.

Tags: Software Development: Revision Control, Implemented in: C, User Interface: Command Line, Role: role::program, scope::utility, Purpose: Editing, use::synchronizing, works-with-format::TODO, Supports Format: works-with-format::diff, works-with-format::plaintext, Works with: works-with::file, works-with::software:source, Text
