NFS support files common to client and server

Use this package on any machine that uses NFS, either as client or server. Programs included: lockd, statd, showmount, nfsstat, gssd, idmapd and mount.nfs.

Tags: System Administration: Filesystem Tool, Implemented in: C, User Interface: interface::commandline, interface::daemon, Networking: Client, Server, Network Protocol: protocol::nfs, role::program
