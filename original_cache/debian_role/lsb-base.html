Linux Standard Base 4.1 init script functionality

The Linux Standard Base (http://www.linuxbase.org/) is a standard core system that third-party applications written for Linux can depend upon.

This package only includes the init-functions shell library, which may be used by other packages' initialization scripts for console logging and other purposes.

Tags: System Administration: System Boot, Implemented in: sh, bash, ksh, tcsh and other shells, Role: Need an extra tag, Plugin
