Config File Handling Library used by BIND

The Berkeley Internet Name Domain (BIND) implements an Internet domain name server. BIND is the most widely-used name server software on the Internet, and is supported by the Internet Software Consortium, www.isc.org. This package delivers the libisccfg shared library used by BIND's daemons and clients to read and write ISC-style configuration files like named.conf and rndc.conf.
