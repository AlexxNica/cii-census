GNU Internationalization utilities for the base system

This package includes the gettext and ngettext programs which allow other packages to internationalize the messages given by shell scripts.

Tags: System Administration: Configuration Tool, Software Development: Internationalization, User Interface: Command Line, Role: role::program, scope::utility, Application Suite: GNU, Purpose: Data Conversion, Works with: Text
