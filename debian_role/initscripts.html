scripts for initializing and shutting down the system

The scripts in this package initialize a standard Debian system at boot time and shut it down at halt or reboot time.

Tags: System Administration: System Boot, Implemented in: sh, bash, ksh, tcsh and other shells, User Interface: interface::commandline, interface::daemon, Role: Program, Scope: Utility, Purpose: Configuration
