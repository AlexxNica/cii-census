Debian configuration management system

Debconf is a configuration management system for debian packages. Packages use Debconf to ask questions when they are installed.

Tags: System Administration: Configuration Tool, Implemented in: Perl, User Interface: interface::commandline, interface::text-mode, X Window System, Role: Program, Scope: scope::utility, suite::debian, Interface Toolkit: Need an extra tag, GTK, uitoolkit::ncurses, uitoolkit::qt, Purpose: Configuration
