Tools to test the reachability of network hosts

The ping command sends ICMP ECHO_REQUEST packets to a host in order to test if the host is reachable via the network.

This package includes a ping6 utility which supports IPv6 network connections.

Tags: System Administration: Monitoring, Implemented in: C, User Interface: interface::commandline, network::scanner, Network Protocol: IPv6, Role: Program, Scope: Utility
