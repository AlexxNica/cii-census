Python interface to Debian's Bug Tracking System

This package provides the debianbts module, which allows one to query Debian's BTS via it's SOAP-inteface and returns the answer in Python's native data types.

Tags: Software Development: Bug Tracking, Implemented in: Python, Role: Program, Application Suite: suite::debian, use::searching, Purpose: Data Visualization, Works with: Bugs or Issues
