utility to detect other OSes on a set of drives

This package detects other OSes available on a system and outputs the results in a generic machine-readable format.

Tags: System Administration: System Installation, Implemented in: sh, bash, ksh, tcsh and other shells, User Interface: interface::commandline, role::program, Scope: Utility, Application Suite: Debian
