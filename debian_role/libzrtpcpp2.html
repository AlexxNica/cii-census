ccrtp extension for zrtp/Zfone support

This library is an extension to the GNU RTP Stack, libccrtp, that offers compatibility with Phil Zimmermann's zrtp/Zfone voice encryption, and which can be directly embedded into telephony applications. The current release is conformant to the ZRTP specification.
