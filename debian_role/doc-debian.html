Debian Project documentation and other documents

The Debian Project is an association of individuals who have made common cause to create a free operating system.

In this package, you will find: 

  * Debian Linux Manifesto,
  * Constitution for the Debian Project,
  * Debian GNU/Linux Social Contract,
  * Debian Free Software Guidelines.

Additionally provided are: 

  * Debian Bug Tracking System documentation, and
  * Introduction to the Debian mailing lists.

All of these files are available at ftp://ftp.debian.org/debian/doc/ and mirrors thereof.
