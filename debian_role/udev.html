/dev/ and hotplug management daemon

udev is a daemon which dynamically creates and removes device nodes from /dev/, handles hotplug events and loads drivers at boot time.

Tags: System Administration: System Boot, Filesystem Tool, Hardware Support, admin::kernel, hardware::detection, Implemented in: C, implemented-in::shell, interface::daemon, Role: Program
